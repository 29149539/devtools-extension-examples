This example extension shows how to use Model View Controller (MVC) pattern in context of an extension for Firefox DevTools. It also demonstrates how to connect all components of the pattern together through events.

Use case:

  1. Clicking on the button below generates standard click event handled by this View (myView.html).
  2. The event is forwarded throug a message manager to Controller living in the chrome scope (myPanel.js) together with current time [ms] as a payload.
  3. When the time arrives to the Controller it's further passed to Model (myExtension.js, also living in the chrome scope) to nicely format it.
  4. The formatted string is sent back to this View (through message manager to the content scope and finally as DOM event to the View)

Checkout the Browser Console for logs.
